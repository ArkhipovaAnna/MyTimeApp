Это приложение позволяет узнать дату, время и часовой пояс определенного населенного пункта по запросу пользователя. 
При написании этого проекта использовались следущие технологии:
- создание пользовательских хуков 
- запросы информации со сторонних сайтов 
- добавление карты от Яндекса
- изменение темы с сохранением значения в localStorage
